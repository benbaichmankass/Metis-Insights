# Hugging Face workflows

Use Hugging Face for datasets, model artifacts, and optional Spaces.

Use names like:

```text
ict-trading-bot-btcusdt-1m
ict-trading-bot-backtest-results
```

## Model naming

Use names like:

```text

```
