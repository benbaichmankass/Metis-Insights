# `tsfm_emb_*` side-stream. TRAINER-SIDE ONLY — the live money-box never installs

# (embedding computed live) is the gated follow-up that would revisit the
# live-VM dependency.

#     (A stable-Python trainer venv would remove this constraint — future vm-ops.)
